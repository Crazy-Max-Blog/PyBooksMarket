Цель: Сделать простой сайт, где можно будет удобно обмениваться книгами.

Части проекта:
* Пользователь: имя, никнейм, информация о себе.
* Книга: название, фотографии
* Лайки: 
  * Пользователь может лайкнуть книгу, но не свою.
  * Пользователь может просматривать, кто лайкнул определённую книгу.
  * Пользователь может просматривать лайки определённого человека
