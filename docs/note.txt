На главной странице есть кнопки "Зарегестрироваться" и "Войти".
После входа кнопки меняются на "Добавить книгу", "Профиль", "Выйти".
Через главную страницу можно искать книги по текстовому запросу
У своей книги есть кнопки "Редактировать", "Редактирровать изображения", "Удалить книгу".
На странице редактирования изображений можно изменить порядок изображений перетаскиванием, удалить изображение;
  Есть кнопка для добавления изображений.
Не свою книгу можно лайкнуть по кнопке на станице этой книги.
На странице книги так же можно посмотреть, кто лайкнул эту книгу.
  При нажатии на имя человека из этого списка откроется его профиль.
В профиле человека можно посмотреть, какие куниги он лайкнул, какие книги у него есть.
