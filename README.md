# Третий проект для Яндекс.Лицея - PyBooksMarket

Сайт для удобного обмена книгами.

Структура проекта:
- docs/ - документация:
  - note.txt - пояснительная записка
  - presentation.pptx - презентация
  - tech_task.md - техническое задание
- \_\_main\_\_.py - главный файл
- requirements.txt - список зависимостей проекта
- templates/ - шаблоны
- forms - формы
- data - модели таблиц базы данных.
- init.sh - скрипт, создающий виртуальное окружение и устанавливающий зависимости
- run.sh - скрипт, запускающий сервер
